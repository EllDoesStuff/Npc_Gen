# Overview 

This is a basic NPC generator that can be incredibly useful in your next text adventure game. Many popular games have NPC's, and at this point it's become a staple of games entirely. This NPC generator generates random NPC's with Damage, Defense, Health, and a special number! All of these options allow for many things to be created, like fights, or companions, or anything, really!
